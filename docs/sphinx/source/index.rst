Dokumentacja AgentMT5
=====================

AgentMT5 to system automatycznego handlu wykorzystujący sztuczną inteligencję do analizy rynku i podejmowania decyzji tradingowych. 
Agent integruje zaawansowane modele AI (Claude, Grok, DeepSeek) z platformą MetaTrader 5, zapewniając autonomiczne zarządzanie 
pozycjami przy zachowaniu ścisłej kontroli ryzyka.
